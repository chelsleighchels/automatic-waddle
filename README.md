# YWAM Ensenada 2026 — Fundraising Tracker

A fundraising tracker for the Cornerstone Fellowship Students Ministry mission trip to Ensenada, Baja California (June 16–23, 2026).

Live site: [www.cfstudentsfundraising.charity](https://www.cfstudentsfundraising.charity)

---

## What It Does

- Displays fundraising progress for each student and staff member
- Shows individual donor breakdowns per person
- Tracks a shared General Team Fund for overflow and undesignated gifts
- Filter by funded / in progress / not started
- Search by name or donor
- Fully responsive (works on mobile)

---

## Tech Stack

- Plain HTML, CSS, and JavaScript (no frameworks)
- Hosted on [Vercel](https://vercel.com)
- Custom domain via `.charity` TLD

---

## Deployment

This site is deployed via Vercel. Any changes pushed to the `main` branch will automatically redeploy the live site.

To update student data or donor information, edit the `PEOPLE` array and `TEAM_FUND` object inside `index.html`.
